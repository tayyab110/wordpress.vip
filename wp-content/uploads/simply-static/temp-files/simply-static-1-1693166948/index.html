X AI Tools Use Case

X AI Tools ran a marketing beta test for Infinite Loop, a project on the Ethereum blockchain. X AI ran for 2 days before the token launch with an AI tweet going out, on average, every 4 minutes with the goal of showing these tweets to potential ETH investors. X AI Tools then ran for a day after the launch of the token had commenced.
The X AI Tweets generated over 18,600 views targeted at potential ETH investors. The X AI platform quote tweeted images, videos and tweets from the main Infinite Loop twitter page in order to market specific messaging about the project. These quote tweets gained over 12,700 views.
